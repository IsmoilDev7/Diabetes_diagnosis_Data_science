# Diabetes_diagnosis_Data_science# 🩺 Diabetes Diagnosis Data Science Project

## 📌 Project Overview

This project focuses on building a data-driven solution for diabetes diagnosis using machine learning techniques. The analysis is based on real-world medical data and inspired by the TAMES AI platform.

## 🎯 Objective

To analyze patient data and build a predictive model that can assist in early detection of diabetes.

---

## 📊 Dataset

The dataset includes medical attributes such as:

* Glucose level
* Blood pressure
* BMI (Body Mass Index)
* Age
* Insulin
* Skin thickness

---

## ⚙️ Technologies Used

* Python (Pandas, NumPy, Scikit-learn)
* Jupyter Notebook
* Matplotlib / Seaborn
* SQL (for data querying)
* Power BI (for visualization)

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Imported dataset from medical sources

### 2️⃣ Data Cleaning

* Handled missing values
* Removed inconsistencies

### 3️⃣ Exploratory Data Analysis (EDA)

* Analyzed feature distributions
* Checked correlations between variables

### 4️⃣ Feature Engineering

* Selected important features
* Scaled numerical data

### 5️⃣ Model Building

* Logistic Regression
* Decision Tree
* Random Forest

### 6️⃣ Model Evaluation

* Accuracy score
* Confusion matrix
* Precision & Recall

---

## 📈 Results

* Achieved high prediction accuracy
* Identified key factors affecting diabetes
* Built a reliable classification model
